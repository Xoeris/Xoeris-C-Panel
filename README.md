# Xoeris C-Panel  
**iOS-Style Control Center Mod for HyperOS**

Xoeris C-Panel transforms the default **HyperOS Control Center** into a clean, modern **iOS-inspired Control Center**, focusing on smooth visuals, refined layout, and better UX consistency and of course without losing HyperOS performance.

Built for Android modders, themers, and UI enthusiasts who want Apple vibes on Xiaomi’s ecosystem.

---

## Features
- iOS-style Control Center layout
- Clean & minimal visual hierarchy
- Polished toggles, sliders, and spacing
- Optimized for HyperOS SystemUI
- Lightweight, no unnecessary background services
- Designed for stability and daily use

---

## Compatibility
- **HyperOS only**
- **Android 15+ supported**
- **Android 16+ required for Dynamic Island support**

> ⚠️ Dynamic Island features will **not** work on Android versions below 16.

---

## Requirements
- HyperOS ROM
- Root access *(Magisk / KernelSU / APatch recommended)*
- SystemUI modification enabled
- Basic understanding of Android modding

---

## Installation
> Detailed installation steps may vary depending on your setup.

General steps:
1. Backup your system (highly recommended)
2. Install via Magisk Module / manual SystemUI replacement
3. Reboot
4. Enjoy iOS-style Control Center

> Always clear SystemUI cache if visual glitches appear.

---

## Known Limitations
- Not compatible with MIUI (non-HyperOS)
- Some OEM-specific features may not be mirrored exactly like iOS
- Animation behavior depends on device & ROM implementation
